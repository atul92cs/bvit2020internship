Normal website with css externally

Why external css is used?

External css makes our code much more readable and clean as the integration of the code and the tags are much more visible and changes can be made easily.

Disadvantage

We have too many files to serve and with more page more css files have to be integrated

So what to use?

Use the css externally mostly with fewer files and different name so that memory will be less occupied
